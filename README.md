# Pi5-Artix-FPGA-Hat
An AMD/Xilinx Artix 50T FPGA on a Pi5 Hat with PCIe and GPIO interconnects as well as SPI programming.

Proposing to use XC7A50T-2FTG256I (LCSC/JLCPCB: C2976033)
